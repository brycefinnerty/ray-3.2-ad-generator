# Ad copy frameworks

Lazy-load this when writing new body/title in Phase 5.5. Pick **one** framework
per variant; the framework dictates structure and voice.

When generating multiple variants, use **different frameworks** so the user can
A/B test angles, not just rewordings.

## Selection guide

Match framework to the image and the brand voice:

| Image / brand cue | Strong matches |
|---|---|
| Visual comparison, before/after, debunking | Challenge Status Quo, First Person Before/After Journey |
| Feature-rich product, packaging-forward | Feature Benefits, Top 3 Reasons |
| Founder / personality brand (e.g. Mr. Paid Social) | First Person, Founders Story, "I saw this on TikTok" |
| Time-sensitive offer | Urgency |
| Bold proof point / numeric claim | Statistic |
| Soft hook, vibes-led | Puns/Teasers, Hook |
| Pain-point empathy | Relatable, IF Statements |
| Social proof leading | Testimonial, First Person Testimonial |

## The frameworks

### First Person
Written in first person to feel like a friend's post. Start with "I". Tailor
phrasing to the target age demo (Facebook in 50s/60s ≠ Instagram in 20s/30s).
Example seed: *"I think a lot of us are guilty of one or two of these..."*

### Feature Benefits
Structure: HOOK → FEATURE BENEFITS (✓ list) → QUOTE → RATING + SAFETY → CTA.
Example: *Shop the best selling wallet of all time and choose from over 30+
colors and styles. ✔️ Ultra-Slim, Minimal Design ✔️ RFID Blocking ✔️ Lifetime
Satisfaction Guarantee. "Slim yet expandable, low profile, but fully featured."
— GQ. 50,000+ ⭐⭐⭐⭐⭐ Reviews. Free Shipping & Returns | 99 Day Trial. Need We
Say More? Find Your Ridge Today.*

### Testimonial
A single golden review in quotes, with attribution + stars.
Example: *"Awesome smelling products with great ingredients! ..." — Alec M.
⭐⭐⭐⭐⭐*

### First Person Testimonial
Testimonial written in first person.
Example: *I kept seeing this Blender all over the internet so I had to try it!
Let me tell you that I was not disappointed! 🥤*

### Urgency
Limited time / last chance / running out / limited quantity.
Example: *🚨 LAST CHANCE TO GET 55% OFF! SALE EXTENDED THROUGH THIS WEEK ONLY!
... Why are women buying 5 of these 100% Mulberry silk pillowcases? ...*

### Statistic
Lead with a number. Structure: STATISTIC HOOK → VALUE PROPS → 3RD PARTY VALIDATION.
Example: *Men can get Generic for Viagra® delivered directly to the door for
95% cheaper than brand name Viagra®. ... As seen in Men's Health, Playboy
and GQ.*

### Top 3 Reasons
Three best reasons why this product wins.
Example: *Bone Broth is a foundational health food! It's... 🌟 A low calorie
gut health essential 👏 An immune system booster 🚀 A great source of protein,
collagen and amino acids. Over 20,000 customers choose ...*

### First Person Before/After Journey
Transformation story in first person, often with a day-by-day timeline.
Example: *I'm 49 and I lost 20 pounds without even trying! ... 🌊 Day 1: ...
🌪 Day 7: ... 💤 Day 14: ... 🔥 Day 21: ... ⚡ Day 28: ... Click below to try
it for yourself.*

### Relatable
Make the reader feel seen and understood. Open with their internal experience.
Example: *Procrastinators are guilt-motivated people because they can't find
the consistent motivation to execute tasks ...*

### Puns/Teasers
Very short, witty.
Example: *Dress less dad and more daddy.*

### Challenge Status Quo
Call out what's broken in the category, then explain the better way.
Example: *Most people in this space will sell you on the idea of a secret
formula, hack, or strategy to get rich. That's the problem with so much of
the education right now ... [why it's wrong] ... [why this product is the
answer] ...*

### Founders Story
First-person, founder-voiced origin / mission.
Example: *"In 2017, I was plagued with severe inflammation ... I turned to
the supplement aisle ... was met with artificial, overly processed and
mislabeled junk ... I decided to build my own plant-based vitamin
alternative ... — Sara, Founder and CEO"*

### "I saw this on TikTok"
First-person social-proof framing.
Example: *Ok so I saw this on Tiktok and let me tell you guys, it did not
disappoint.*

### Hook
Use the hook from a video as the primary text.
Example: *The Teeth Whitening Hack that's been liked over 17 MILLION times
TikTok 🤯*

### IF Statements
Open with a compelling "If you ___" conditional.
Example: *If you're tired of living paycheck to paycheck, it might be time
to pull the trigger on that degree.*

## How to use this in Phase 5.5

1. Read the new image carefully — what's the primary visual hook?
2. Read the cloned ad's body/title to feel the brand voice.
3. Pick the framework that **aligns with both** the image and the voice.
4. Write tight: title ≤ 40 chars, body ≤ 125 chars (Meta truncation floor).
5. State which framework you used so the user can A/B by structure later.
6. For N>1 variants, use **different** frameworks — don't reword the same one.
